# BunnyPlayerDemo

Simple iOS 13+ UIKit demo (Swift 5) with:
- Home screen with 2 buttons
- Multilanguage video player
- Advertisement + main video flow using VMAP parsing

## Run

```bash
pod install
open BunnyPlayerDemo.xcworkspace
```

## Project Structure

- `Podfile`
- `BunnyPlayerDemo.xcodeproj`
- `BunnyPlayerDemo/AppDelegate.swift`
- `BunnyPlayerDemo/SceneDelegate.swift`
- `BunnyPlayerDemo/HomeViewController.swift`
- `BunnyPlayerDemo/MultiLanguagePlayerViewController.swift`
- `BunnyPlayerDemo/AdvertisementPlayerViewController.swift`
- `BunnyPlayerDemo/Info.plist`
